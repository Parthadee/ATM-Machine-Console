A simple console-based ATM simulation project developed in Java using Object-Oriented Programming concepts. The system allows users to perform basic banking operations such as balance inquiry, withdrawal, deposit, and transaction history.

---
## Features

With this ATM program, a user can:

- Deposit money
- Withdraw money
- Check account balance
---
## Requirements

To run this project, you need:

- **Java 8 or later**

You can run this project in:

- Eclipse
- IntelliJ IDEA
- NetBeans
---
## How to Run the Project

Follow these steps:

1. Clone or download this repository to your computer.

2. Keep these files in the same folder:

   - `ATM.java`
   - `Account.java`
   - `OptionMenu.java`

3. Open the folder in your Java IDE  
   or open **Command Prompt / Terminal**.

4. Go to the project folder:

   ```bash
   cd ATM-Machine

- **Compile the Program:**
```bash
javac ATM.java
---
After compiling, some .class files will be created.
Run the program:
```bash
java ATM
